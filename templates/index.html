What does this tool do?

  • Converts Bambu Lab .3mf projects to Snapmaker U1
  • Applies the 0.20mm Standard print profile
  • Remaps colors and filament types to U1 compatible profiles
  • Enables Tree Supports (auto) if the original model has supports enabled

Note: The converted file should be opened in Snapmaker Orca for final slicing.
